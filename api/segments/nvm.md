## prompt_nvm
*Display the current Node version from NVM. *

#### Arguments

- **$1** (string) Alignment - left | right
- **$2** (integer) Segment index
- **$3** (boolean) Whether the segment should be joined


#### Notes

*Only prints the segment if different than the default value. *
